## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation Guide

```bash
# Clone the repository
git clone <your-repo-url>
cd dormitory-management-nextjs

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
```

- Configure your institution settings
